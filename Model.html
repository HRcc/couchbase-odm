getMulti(ids, optionsopt) → {Promise.<Object>}

getMulti

if at least one of ids is found, no Error is throwed. Instead, if a operation fails to get a id from ids array, a StorageError is returned in place of resulting value in returned collection.

if no id is found, the promise is rejected with StorageMultiError which provides array of specific errors for each document undererr.errors.

Returned list is in format where index is an id (not whole document's key) and value is either instance of Instance or StorageError.
